# 🆕 INSTALADOR DE PYTHON 3.10 + DEPENDENCIAS

Para poder ejecutar los programas, uPLC requiere tener instalado en nuestros equipos Python y sus librerias (dependencias).
Para no tener errores, seguí estos pasos para instalar y ejecutar uPLC correctamente:

## 🚀 Instalación paso a paso:
1. ✅ Necesitamos conexion a INTERNET por unica vez...
2. ✅ Descarga el ZIP ("INSTALADOR.zip") y descoprimir en cualquier carpeta. Se recomienda evitar rutas con espacios o caracteres especiales.
3. ✅ Click en el archivo BAT ("INSTALAR.bat") e iniciara la instalacion automaticamente. El proceso puede tardar unos minutos.
No cierres la ventana hasta que aparezca el mensaje de finalización...

a medida que va avanzando el instalador y termina cada fase, vamos presionando cualquier tecla para continuar. Al finalizar, tendremos el siguiente mensaje:

### Este proceso:
- Configura el entorno portable de Python
- Instala automáticamente las dependencias necesarias
- Deja el sistema listo para usar

4. ✅ Al finalizar, ya tenemos todo lo que necesitamos para poder ejecutar uPLC en sus distintas versiones... Descargamos la version que queremos, guardamos en cualquier carpeta, descomprimimos y al ser una aplicacion portable, la podemos ejecutar desde cualquier lugar, incluso desde un pendrive.

---

## ⚠️ Notas importantes
- Es normal que aparezcan advertencias de pip (no afectan el funcionamiento).
- No es necesario tener Python instalado previamente.
- Si Windows muestra advertencias de seguridad, seleccionar “Ejecutar de todas formas”.

## 🛠️ Solución de problemas
- El instalador se cierra solo: ejecutar como administrador
- Error de dependencias: volver a ejecutar INSTALAR.bat
- Pantalla en blanco o error al iniciar: eliminar la carpeta y reinstalar
